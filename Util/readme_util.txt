The provided utilities allow to work with certain cartridge's features:

\c2man.com		- multi-purpose utility for most of MSX2 and later computers
\c2man40.com		- multi-purpose utility for most of MSX1 computers
\c2edit.com		- directory editor for most of MSX2 and later computers
\c2edit40.com		- directory editor for most of MSX1 computers
\c2sram.com		- utility to back up and restore the emulated FMPAC's SRAM data
\c2ramldr.com		- utility to load ROMs into cartridge's shadow RAM (MegaROM-like)
\c2backup.com		- utility to back up and restore contents of the FlashROM chip
\c2cfgbck.com		- utility to back up and restore contents of the configuration EEPROM chip
\c2dsktst.com		- utility to test DISK controller read/write functionality
\c2finder.com		- utility to detect Carnivore family cartridges using two different methods
\c2fwupd.com		- utility to update FPGA firmware (with FIRMWARE.BIN file)
\Special\		- folder for special utilities to be used on Korean and Arabic MSX computers

The following utilitues are only available for authorized manufacturers:

\c2bforce.com		- utility to back up and restore contents of the FlashROM chip without security checks (Caution!)
\c2cforce.com		- utility to back up and restore contents of the configuration EEPROM chip without security checks (Caution!)
\c2tester.com		- utility to test various components of the cartridge (Caution!)

Please check the readme.txt file for the description of the utilities.
